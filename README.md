# gmail_scripts
Google App Scripts for Gmail

This project is available [here](https://script.google.com/u/1/home/projects/11w6QB5Qk16JNg6IQL6s3ZPw0Ns-qGnVSX3WPpLecRpVuKWvuq2YeHo5x), 
where you can set up triggers for your own account. However, you will not be able to modify the scripts or view logs. Follow the project 
setup instructions to set up your own project, where you will have full control.

# Quickstart

1. Open [project](https://script.google.com/d/11w6QB5Qk16JNg6IQL6s3ZPw0Ns-qGnVSX3WPpLecRpVuKWvuq2YeHo5x/edit?usp=sharing)
2. Navigate to `Triggers` from the left side-bar. 
3. Click `Add Trigger`
4. Choose function(s) to run and set up cron configuration.
5. Click `Save`
